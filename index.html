Gutenweb

Gutenweb is a typographical stylesheet. It is a CSS file ready to use for quick document generation in html (and epub).

Gutenweb aims to forgot styling when marking-up a document, as you do when you generate a document in LaTeX. It is suitable for not too complex documents.

It adds some default styles which prioritize readability and they adapt to screen or print media. A lot of the concepts are taken from The Elements of Typographic Style Applied to the Web.

This web page is using Gutenweb (plus, if your rendering engine supports it, background strips to show the vertical rhythm used), so you are right know seeing the look&feel generated. Print to a PDF to look at the paper version.

Setup

Get a copy in the official repository or download the zip.

Just copy dist/gutenweb.css to your css folder and add a link to it:

<link rel="stylesheet" href="gutenweb.css">

Special formats

.toc for table of contents

To create a table of contents, mark-up a table with .toc as its class. Put the section titles inside <th> and its page numbers inside <td>:

Example:

 <table class="toc">
   <tr>
     <th>Section 1</th><td>3</td>
   </tr>
   <tr>
     <th>Section 2</th><td>6</td>
   </tr>
 </table>

Looks:

Section 13
Section 26

.note for notes

Example:

<p class="note">I am a small note</p>

Looks:

I am a small note

.chapter to force a page break

Add .chapter class to force a page break, for example in a heading.

<h2 class="chapter">New chapter</h2>

.caps and .nums for uppercase and numeric strings

Uppercase and numeric strings usually deserve a small increase in its letter-spacing.

Example:

<p><span class="caps">UPPERCASE STRING</span></p>
<p><span class="num">936543476</span></p>

Looks:

UPPERCASE STRING

936543476

pre with .code for block level code

For inline code, use simple <code> tag. For block-level code, wrap it in a <pre> tag with a .code class.

Example:

 <pre class="code">
   <code>
     &lt;p&gt;Hello World!&lt;/p&gt;
   </code>
 </pre>

Looks:

<p>Hello World!</p>

.verse for verses

Example:

<pre class="verse">Is not Gutenweb
such a nice
stylesheet?</pre>

Looks:

Is not Gutenweb
such a nice
stylesheet?

Internals

The stylesheet is written in Sass and compiled to CSS. The main file is src/gutenweb.sass, which use src/_print.sass, where there are print styles, and the helper file src/_helper.sass, where there are some usefull functions and mixins.

Eric Meyer reset stylesheet is also used to avoid problems with default rendering engines styles.

This page is written with slim template language (the examples/index.html.slim file) and compiled to html (the examples/index.html file).

Release Policy and version notes

Gutenweb uses semantic versioning system. Now it is still at initial development.

It is sure that there are some common mark-up pattern that it is not covered. Please, open an issue for any error or improvement you can suggest. It will be added as soon as possible.
